# tmdb-app
TMDB Android Application

## Description
This is a simple Android application that uses the TMDB API to display a list of movies and their details.
Using:
- Kotlin
- MVVM
- Retrofit
- Room
- Coroutines
- Hilt
- Compose

## Installation

Add api key to local.properties file
```
api_key="YOUR_API_KEY"
```
